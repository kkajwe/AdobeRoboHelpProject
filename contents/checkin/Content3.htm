ACTION 

 Enter or "pick" one of the following codes:                
                                                            
   C - to list all transactions involving cash and checks   
       received or refunded                                 
                                                            
   V - to list all voucher prepayments                      
                                                            
   L - to list all rentals where loyalty points were        
       redeemed (Loyalty points appear at car return only.) 
                                                            
   T - to list all tax exempt rentals                       
                                                        
   N - to list all transactions where coupons were applied  
                                                            
  NU  - to list transactions with upgrade coupons           


  NM - to list transactions with money off coupons        
                                                            
  NT - to list transactions with free time/days coupons   
                                                            
  ND - to list transactions with discount coupons         
                                                        
  NR - to list transactions with reusable coupons         
                                                            
 Add a one-letter counter identifier to the ACTION codes    
 to display the lists by counter (if appropriate)           
                                                            
 Example:  CA (cash/check transactions for counter A)
